Boxwood · Layout Engine

One Tree In, Live Layout Out

A single declarative tree (cols / rows / gap / padding) is resolved into absolute {x, y, w, h} boxes and drawn as SVG. Because you own the coordinates, the bezier connectors are trivial to route. Drag the corner — the same tree re-resolves and the whole composition reflows.

No build step — this page loads @canwork/boxwood from a CDN and calls resolveLayout() on every resize. The whole integration is the drawScene() function in this file's source. For the TypeScript version, see the *.scene.ts files and run npx tsx examples/build.ts.
