Boxwood — One Tree In, Live Layout Out

A single declarative tree (cols / rows / gap / padding) is resolved into absolute {x, y, w, h} boxes and drawn as SVG. Because you own the coordinates, the bezier connectors are trivial to route. Drag the ◢ corner — the same tree re-resolves and the whole composition reflows.

◢
This page has no build step — it loads @canwork/boxwood from a CDN and calls resolveLayout() on every resize. View source: the entire integration is the drawScene() function. To see how you'd wire this into a real TypeScript project, read the *.scene.ts files and run npx tsx examples/build.ts.
